# fundamental-booster.py

📘 Fundamental Booster – Interactive Personal Data Collector

📌 Overview

Fundamental Booster – Interactive Personal Data Collector is a beginner-friendly Python project that collects personal information from the user, performs simple calculations, demonstrates type conversion, and displays the data type and memory address of variables. It is designed to help beginners understand Python fundamentals through an interactive console application.

---

✨ Features

- Collects user information (Name, Age, Height, Favourite Number)
- Calculates the approximate birth year
- Converts height from meters to centimeters
- Performs arithmetic operations
- Demonstrates type casting ("int", "float", "str")
- Displays variable data types using "type()"
- Displays memory addresses using "id()"
- Uses string concatenation and f-strings
- Prints a formatted summary table

---

🛠 Requirements

▶️ How to Run

1. Install Python 3.x.
2. Save the file as "personal_data_collector.py".
3. Open a terminal or command prompt.
4. Run the program:

python personal_data_collector.py

5. Enter your details when prompted.
